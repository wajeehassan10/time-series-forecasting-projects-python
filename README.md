# Time-Series Forecasting Projects with Python

This repository contains forecasting projects completed as part of my Engineering & Management studies at Technische Hochschule Ingolstadt.

The project applies time-series analysis and forecasting methods to two datasets:

1. A self-collected recovery dataset
2. A public Bitcoin price dataset

## Project Overview

The aim of this project was to understand how different forecasting models perform on different types of time-series data. The work includes preprocessing, stationarity testing, trend and seasonality analysis, model training, and forecast evaluation.

## Methods and Models Used

- Time-series preprocessing
- Stationarity testing with ADF test
- Trend and seasonality analysis
- Naive Forecast
- ARIMA
- Prophet
- Holt-Winters
- LSTM Neural Network
